# Personal Portfolio Web Application

## Project Description

This is a personal portfolio website built as part of a software engineering course. It showcases projects, provides a contact form, displays live weather and inspirational quotes, and demonstrates modern web development best practices including accessibility, responsive design, and state management (light/dark mode, personalized greeting, etc).

## How to Run Locally

1. Clone the repository:
   ```
   git clone https://github.com/Abdulrahman-Alhendawi/assignment-4.git
   ```
2. Open the `assignment-4/assignment-4` folder in your code editor.
3. Start a local web server in the project directory. You can use:
   - **VS Code Live Server extension** (recommended)
   - Or Python‚Äôs built-in server:
     ```
     python -m http.server
     ```
4. Open your browser and go to `http://localhost:8000` (or the port shown in your terminal).
